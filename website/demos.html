⚡
How it works: Box2D's C17 source is compiled to a ~150KB WASM module using Zig's cross-compiler. Each frame, PNGine calls physics_step(dt) via #wasmCall, reads body transforms into a GPU storage buffer, and draws everything with instanced rendering.
